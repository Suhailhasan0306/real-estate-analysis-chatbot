# real-estate-analysis-chatbot

A full-stack assignment project built with **Django (REST API)** and **React**.

## Features

- Chat-style interface to ask queries like:
  - "Analyze Wakad"
  - "Compare Ambegaon Budruk and Aundh demand trends"
  - "Show price growth for Wakad over the last 3 years"
- Metric toggle:
  - **Price (total sales)**
  - **Demand (total sold)**
- Backend reads data from an Excel file (`backend/data/real_estate.xlsx`)
- Trend line chart + table view for yearly data

## Tech Stack

- **Backend:** Django, Django REST Framework, pandas
- **Frontend:** React (Vite), Axios, Recharts, Bootstrap
- **Deployment:** Render (Python Web Service + Static Site)
